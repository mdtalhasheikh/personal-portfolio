## Mohammad Talah Sheikh Portfolio

A professional portfolio website built with Next.js, using content from the LinkedIn profile export in `Profile-2.pdf`.

### Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.

The Digital Twin chat uses OpenRouter from the server-side API route. Set either `OPENROUTER_API_KEY` or `OPENAI_API_KEY` in `.env`.
